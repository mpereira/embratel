* embratel
** Instalando
#+BEGIN_SRC
gem install embratel
#+END_SRC

** Para rodar os testes
*** Pegue o código
#+BEGIN_SRC
git clone git://github.com/murilasso/embratel.git
cd embratel
#+END_SRC

*** Instale o bundler e as dependências (se elas já não estiverem instaladas)
#+BEGIN_SRC
gem install bundler
bundle install
#+END_SRC

*** Rode os testes
#+BEGIN_SRC
rake test
#+END_SRC
