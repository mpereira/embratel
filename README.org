* embratel
** Instalando
#+BEGIN_SRC
gem install embratel
#+END_SRC
** Testando
   Para rodar os testes você precisa ter o
   [[http://github.com/carlhuda/bundler][bundler]] instalado.
#+BEGIN_SRC
gem install bundler
bundle install
rake test
#+END_SRC
